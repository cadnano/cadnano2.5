.. cadnano shortcuts

Keyboard Shortcuts
==================

::

  Document:
    New           ⌘N
    Open          ⌘O
    Close         ⌘W
    Save          ⇧⌘S
    SaveAs        ⌘W
  
  Tools:
    CreateTool    C
    InsertTool    I
    BreakTool     B
    SkipTool      K
    AddSequence   A
  
  Filters:
    Helix         H
    Endpoint      E
    Crossover     X
    Scaffold      S
    Staple        T
    Fwd           W
    Rev           R
  
  Views:
    Move          ←↑↓→
    Zoom to fit   F


View shortcuts only work when the view has focus.
